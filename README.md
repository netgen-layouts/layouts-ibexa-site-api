# Netgen Layouts & Ibexa Site API integration

## Installation instructions

### Use Composer to install the integration

Run the following command to install Netgen Layouts & Ibexa Site API
integration:

```bash
composer require netgen/layouts-ibexa-site-api
```

Symfony Flex will automatically enable the bundle.

## Running tests

Running tests requires that you have complete vendors installed, so run
`composer install` before running the tests.

You can run unit tests by calling `composer test` from the repo root:

```bash
$ composer test
```
